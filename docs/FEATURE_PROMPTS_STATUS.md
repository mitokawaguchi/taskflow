## 環境変数（プロンプト補足との対応）

| 変数 | プロンプトでの想定 | 本リポジトリの状況 |
|------|---------------------|---------------------|
| `VITE_SUPABASE_URL` | Supabase | ✅ `.env.example` にあり（既存アプリで利用） |
| `VITE_SUPABASE_ANON_KEY` | Supabase | ✅ 同上 |
| `CHATWORK_API_TOKEN` | Chatwork 連携 | ⬜ `.env.example` 未記載（1-2 / 2-1 / 3-4 で必要になる想定） |
| `CHATWORK_MY_ACCOUNT_ID` | 同上 | ⬜ 同上 |
| `CHATWORK_MY_ROOM_ID` | マイチャット DM（1-2） | ⬜ 同上 |
| `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET` / `GOOGLE_REDIRECT_URI` | Gmail OAuth（1-3） | ⬜ 同上 |
| `GEMINI_API_KEY` | Gemini（Phase 3） | ⬜ 同上 |

**運用ルール:** クライアントに載せたくないキーは `VITE_` を付けず、Edge Functions やサーバー側のみに置く（プロンプトの意図に合わせる）。
