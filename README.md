# TaskFlow

個人用タスク管理アプリ。React + Vite で構築。LocalStorage でデータ保存。

## 機能

- カード形式のタスク管理（重要度別カラーコーディング）
- プロジェクト/クライアント別ルーム（進捗ダッシュボード付き）
- 期限設定・ソート（重要度 / 期限 / 作成日 / 名前）
- テンプレート機能
- 朝イチ強制確認モーダル
- ブラウザ通知 + カスタムバイブレーション（PWA対応）

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
npm run preview
```
